# Shared Framework

## newtype AI Stack 七层产业地图

从下到上:

1. **能源层 Energy**: 发电、核电、天然气、可再生能源、电网、PPA、能源调度。
2. **基础设施层 Infrastructure**: 数据中心建设、机柜、液冷、电源、机械工程、配电。
3. **互联与存储层 Interconnect & Memory**: HBM、DRAM、NVLink/InfiniBand、光模块、网卡、存储。
4. **算力硬件层 Compute Hardware**: GPU、ASIC、TPU、AI SoC、加速卡、推理芯片。
5. **算力服务层 Cloud / IaaS**: AWS、Azure、GCP、Oracle、CoreWeave、Nebius、Lambda 等算力租赁和云服务。
6. **模型层 Models**: OpenAI、Anthropic、Google、DeepSeek、Llama、foundation models、模型 API。
7. **应用层 Apps**: ChatGPT、Copilot、Cursor、垂直 SaaS、企业 AI 工具、消费者 AI 应用。

核心关系:

- 上层是下层的客户,但稀缺性常集中在下层。
- 2024-2027 周期中,投资稀缺性优先看第1-4层。
- 第7层最贴近用户,但常受模型降价和"算法成本归零"挤压。

## 距离终端算力需求

- 第4层算力硬件: 距离 0,需求最直接。
- 第3层互联存储、第5层算力服务: 距离 1。
- 第2层基础设施、第6层模型: 距离 2。
- 第1层能源、第7层应用: 距离 3。

距离 0-1 的公司验证速度快、市场关注高;距离 2-3 的公司传导更慢,但可能产生错杀或滞后机会。

## 卖铲子定律

真正稳定赚钱的往往不是挖金子的人,而是给挖金子的人卖铲子的人。

- 第1-4层: 通常是卖铲子,优先研究。
- 第5层: 也是卖铲子,但集中度高、利润率结构取决于云厂商竞争。
- 第6层: 介于卖铲子和挖金子之间;模型 API 像铲子,模型公司自身可能持续烧钱。
- 第7层: 多数是挖金子,赢家通吃且可替代性更高。

## 稀缺性等级

- **核心稀缺**: 全球 1-3 家能做,短期难追赶,议价能力强。例如 EUV、先进制程、顶级 HBM。
- **次级稀缺**: 少数玩家能做,但有第二梯队追赶。例如高端 GPU、超大规模云、关键网络芯片。
- **可替代**: 玩家多、门槛中等、价格战概率高。例如普通服务器组装、通用应用 SaaS。

## 跟踪分级

- **核心跟踪**: 高稀缺 + 卖铲子 + 距离 <= 1,可能成为组合核心。
- **战术跟踪**: 次级稀缺 + 卖铲子 + 距离 <= 2,关注财报、订单、价格信号。
- **观察池**: 距离较远或稀缺性弱,等待错杀、催化或验证信号。
- **过滤掉**: 应用层挖金子 + 可替代 + 缺乏独立验证,不投入主要研究精力。

## 合规与数据边界

所有输出都是基于方法论的研究分析,不是个性化投资建议。不要直接替用户做买卖决定。具体投资决策应结合个人风险承受能力、资金属性、税务状况,并咨询合格投资顾问。

实时事实必须验证,包括但不限于股价、市值、估值倍数、财报、订单、市场份额、产能、合同、监管审批和管理层指引。
